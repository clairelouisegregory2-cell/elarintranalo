# Élarin Translator

A fantasy language translator and lexicon for **Élarin**, the Language of Memory, Destiny, and Becoming.

## About

Élarin is a constructed language built around themes of:

* Memory
* Destiny
* Truth
* Wisdom
* Sacred symbolism
* Mythic philosophy

This project provides a simple web-based translator that can translate between English and Élarin vocabulary.

## Features

* Élarin → English translation
* English → Élarin translation
* Fantasy-inspired user interface
* Responsive design
* Single-file HTML application
* No installation required
